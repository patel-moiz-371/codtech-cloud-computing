# Task 1: Cloud Storage Setup

## Project Overview
[cite_start]This project demonstrates the creation and configuration of secure cloud storage utilizing Amazon Web Services (AWS) Simple Storage Service (S3).

## Implementation Steps
1. [cite_start]**Bucket Creation**: Created a globally unique S3 bucket in the Europe (Stockholm) `eu-north-1` region[cite: 24].
2. **Security Settings**: Kept "Block Public Access" enabled to ensure data privacy and prevent unauthorized internet exposure.
3. [cite_start]**Data Upload**: Uploaded a sample text object (`test-file.txt`) to the bucket root.
4. [cite_start]**Access Control**: Verified that direct Object URL access results in an `Access Denied` error, confirming that permissions are configured securely[cite: 26].

## Deliverables
- [cite_start]**Sample File**: `test-file.txt`.
- [cite_start]**Screenshots**: Proving successful bucket deployment and permission verification[cite: 25, 26].
